#二叉树

二叉树的遍历

二叉树分为前序，中序，后序遍历和层次遍历

递归遍历

java
List<Integer> res = new ArrayList<>();
public List<Integer> order(TreeNode root) {
	recursion(root);
	return res;
}

//前序模式
public void recursion(TreeNode root) {
	if(root == null) return;
	res.add(root.val);
	if(root.left != null) recursion(root.left);
	if(root.right != null) recursion(root.right);
}

//中序模式
public void recursion(TreeNode root) {
	if(root == null) return;
	if(root.left != null) recursion(root.left);
	res.add(root.val);
	if(root.right != null) recursion(root.right);
}

//后序模式
public void recursion(TreeNode root) {
	if(root == null) return;
	if(root.left != null) recursion(root.left);
	if(root.right != null) recursion(root.right);
	res.add(root.val);
}

迭代遍历

前序遍历，中->左->右

用栈来存储node，栈是后入先出的，所以要先放root.right再放root.left

java
List<Integer> res = new ArrayList<>();
stack.push(root);
while(stack.size() > 0) {
	res.add(root.val);
	if(root.right != null) stack.push(root.right);
	if(root.left != null) stack.push(root.left);
}

中序遍历，左->中->右

按照遍历的规则

  1. 一直找左节点直到为空为止

  2. 然后取出最左边的节点后找它的右节点

  3. 重复1的步骤

java
List<Integer> res = new ArrayList<>();
TreeNode node = root;
while(stack.size() > 0 || node != null) {
	if(node != null) {
		stack.push(node);
		node = node.left;
	} else {
		TreeNode top = stack.pop();
		res.add(top.val);
		node = node.right;
	}
}

后序遍历

顺序是 左->右->中，可以看出它是中->右->左反过来，所以先用中->右->左，最后再反转数组就行

java
List<Integer> res = new ArrayList<>();
stack.push(root);
while(stack.size() > 0) {
	res.add(root.val);
	if(root.left != null) stack.push(root.left);
	if(root.right != null) stack.push(root.right);
}
Collections.reverse(res);
